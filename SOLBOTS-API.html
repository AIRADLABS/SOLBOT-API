Solbot API

dev.solbots.io  api.solbots.io

(aka SOLBOT-API)

Rather than using an SDK, it is possible to interact with the network directly through the API.

The Solbot API is an application programming interface that allows developers in the cryptoverse to anonymously store and access encrypted off-chain data files. The data can be used to power any metaverse, game, dapp, or defi development regardless of its native blockchain ecosystem or programming language.

The following is more details about newMint and newPod commands.

They are the internal system commands that are being executed by this bypass.

newMint

Executed every time we mint a new Worker Bot on the Solana blockchain.

newMint creates a registry item on the solbots.io network known as an “offline pod”.

The offline pods have not been purchased yet, and there’s no associated pod address.

The new pod registry item is associated with the NFT via it’s address, mint authority, and signature.

This is what a “offline pod” registry item looks like:

2. Create a new folder

Now that we have our new pod address and key we can create a new folder in the pod.

*Your new folder name can only include letters, numbers, and underscores.

http://dev.solbots.io/newFolder/0-3/?payload=

{

  "pod_address" :  "solbot.a25hNHVWclgvS1gxTnBlaURZeFJUSFRWWlR0Y0JjYkpaazd2ZHFFWFR0MD06OkplZt864RRPtM5MDFKjlLQ=",

  "pod_key" :  "solbot.1632608455.9232.5be0800a5a5eec63f11936d206c9bf235b567d010eef2e0075de635080dc64b1",

  "pod_folder" :  "folder_1"

}

Expected Response

{

  "status": "success",

  "message": "New folder created! (folder_1)"

}
